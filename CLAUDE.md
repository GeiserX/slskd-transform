# CLAUDE.md — slskd-transform

## Overview
Bulk-upgrades a music library from lossy to lossless FLAC by searching the Soulseek network through slskd. Matches songs by audio duration rather than filenames, and includes a rename command for post-download organization.

## Tech Stack
- Python 3.10+
- slskd API integration
- Docker support
- pytest for testing
- YAML configuration (`config.example.yml`)

## Development
```bash
pip install -e ".[dev]"
pytest

# Run search
export SLSKD_API_KEY="your-key"
slskd-transform search --input ./music --recursive

# Run rename
slskd-transform rename --input ./downloads
```

## Architecture
- `src/slskd_transform/` — main package
- `main.py` — CLI entrypoint
- `rename-files.py` — standalone rename utility
- `config.example.yml` — example configuration
